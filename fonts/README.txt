字体文件夹使用说明
==================

如何添加平方公子体字体：
----------------------

1. 下载字体文件
   - 访问字体下载网站（如：字由、字体天下、求字体网）
   - 搜索 "平方公子体" 或 "PingFangGongZiTi"
   - 下载 .ttf 或 .otf 格式的字体文件

2. 重命名字体文件
   - 将下载的字体文件重命名为：PingFangGongZiTi.ttf
   - 注意大小写要和代码中的完全一致

3. 放置字体文件
   - 将重命名后的字体文件放到此文件夹（fonts/）
   - 游戏会自动检测并使用该字体

4. 验证字体
   - 运行游戏，查看控制台输出
   - 如果看到 "成功加载自定义字体: PingFangGongZiTi.ttf" 说明加载成功

推荐的字体下载网站：
------------------
- 字由网：https://www.hellofont.cn/
- 字体天下：https://www.fonts.net.cn/
- 求字体网：https://www.qiuziti.com/
- 素材集市：https://www.sucaijishi.com/

注意事项：
---------
⚠️ 请确保下载的字体是"免费可商用"版本
⚠️ 下载前请仔细阅读字体授权协议
⚠️ 商业项目尤其需要注意版权问题

如果找不到平方公子体：
---------------------
可以考虑使用以下替代字体（都是免费可商用）：
- 站酷高端黑（强烈推荐）
- 清松手写体
- 思源黑体
- 庞门正道粗书体

详细字体使用指南请查看项目根目录的 FONTS_GUIDE.md
